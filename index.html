混合机甲战斗系统 - 左右手武器独立操作

👆 点击游戏区域，然后点击选择游戏模式 → 点击武器配置 → 点击开始游戏

控制说明：

  • 点击选择 - Boss战模式 | 武器配置
  • 点击开始游戏按钮 - 完成配置后开始游戏
  • WASD 或 方向键 - 移动
  • 鼠标左键 - 左手武器（默认：枪）
  • 鼠标右键 - 右手武器（默认：剑）
  • Q键 - 隐藏机能（默认：脉冲护盾）
  • 空格键 - 闪避（朝移动方向快速移动0.15秒，闪避和冷却期间无法攻击）
  • R - 重装武器（可选）
  • F - 切换锁定模式（软锁→硬锁→手动锁）
  • C - 硬锁模式下切换目标
  • ESC/P - 暂停游戏
  • 🎯 三种锁定模式：软锁（自动锁最近敌人）、硬锁（锁定单个敌人）、手动锁（朝鼠标方向攻击）
  • 🔫 自动步枪：自动重装 + 高射速2.2/秒 + 预瞄
  • ⚔️ 脉冲光束军刀：高伤害20点 + 冲刺攻击
  • 🚀 8连导弹发射器：强追踪1.1秒 + 范围爆炸 + 3点伤害
  • 🛡️ 脉冲护盾：70%伤害减免 + 15秒持续 + 40秒冷却
  • 🤖 Boss血量：200点 | 闪避：60%
  • 💨 Boss会智能闪避攻击（青色虚线边框）
  • 🔥 Boss战模式：连续Boss挑战，适合练习技巧
